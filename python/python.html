    • 21. Object-Oriented Programming (OOP)

      Classes, objects, inheritance, etc.

      class Server:
    def __init__(self, hostname):
        self.hostname = hostname

    def reboot(self):
        print(f"Rebooting server: {self.hostname}")

web_server = Server("web01.example.com")
web_server.reboot()

    • 22. Concurrency and Parallelism

      Threads, processes, asyncio.

      import threading
import time

def task(n):
    time.sleep(n)
    print(f"Task {n} finished")

thread1 = threading.Thread(target=task, args=(2,))
thread2 = threading.Thread(target=task, args=(1,))
thread1.start()
thread2.start()
thread1.join()
thread2.join()
print("All tasks done.")
