Bash Scripting Guide

This page covers commonly used statements and loops in Bash scripting with examples.

1. Variables

#!/bin/bash
name="John"
echo "Hello, $name!"

2. Conditional Statements

if Statement

if [ $age -ge 18 ]; then
  echo "You are an adult."
fi

if-else Statement

if [ $age -ge 18 ]; then
  echo "Adult"
else
  echo "Minor"
fi

if-elif-else Statement

if [ $score -ge 90 ]; then
  echo "A grade"
elif [ $score -ge 75 ]; then
  echo "B grade"
else
  echo "C grade"
fi

3. Case Statement

read -p "Enter a number: " num
case $num in
  1) echo "One";;
  2) echo "Two";;
  *) echo "Other number";;
esac

4. Loops

for Loop

for i in {1..5}
do
  echo "Iteration $i"
done

while Loop

count=1
while [ $count -le 5 ]
do
  echo "Count: $count"
  ((count++))
done

until Loop

num=1
until [ $num -gt 5 ]
do
  echo $num
  ((num++))
done

5. Functions

greet() {
  echo "Hello, $1!"
}

greet "Alice"

6. Arrays

arr=("apple" "banana" "cherry")
echo ${arr[1]}
for item in "${arr[@]}"; do
  echo $item
done

7. Reading User Input

read -p "Enter your name: " name
echo "Hello, $name!"

8. Command Line Arguments

echo "Script name: $0"
echo "First argument: $1"
echo "Second argument: $2"
